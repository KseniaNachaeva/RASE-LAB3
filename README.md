# RaceSimulator
Задача 2.5

Симулятор автомобильных гонок. 

Гонки – смоделировать гонки. Реализовать классы – Болид, Механик, интерфейс – погрузчик. События – Стерлись покрышки – заезд в боксы смена колес, Столкновение (с некоторой долей вероятности) – приезжает погрузчик.

## Что умеет

- Добавляй любое количество гонок кнопкой **«+ Add Race»**, каждая работает независимо.
- Запускай все сразу (**«▶ Start All»**) или каждую отдельно кнопкой **Start**.
- Останови или удали гонку в любой момент.

## Как устроена гонка

- 3–6 машин стартуют с одной линии, у каждой своя случайная скорость.
- Шины изнашиваются со временем — чем сильнее износ, тем медленнее машина.
- Когда шины стёрты до 100% — машина едет в боксы, **механик** меняет колёса (2–3 с).
- С некоторой вероятностью машина может врезаться в ограждение — тогда вызывается **погрузчик**, который приезжает на место аварии и эвакуирует болид в боксы (3–7 с).
- Первая машина, проехавшая все круги — победитель.
